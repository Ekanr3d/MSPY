mSpy With Jailbreak

  • Compatible with iOS 6 – 8.3
  • The iPhone or iPad must be connected to the Internet.
  • The monitored device must be jailbroken.
  • You need physical access to the device to install mSpy.

Supported Features:

  • Call Logs
  • Emails
  • Geo-Fencing
  • GPS Location Tracking
  • SMS Messages
  • Browser History
  • Calendar Activities
  • Chat Apps: Snapchat, Skype, Social Networks, Viber & WhatsApp
  • Call & Website Blocking
  • Device Wipeout
  • & more!

mSpy Without Jailbreak

  • Compatible with all iOS versions.
  • The device must be internet-enabled.
  • You do not need physical access to the device if you already have its iCloud credentials.
  • Still, the access may become necessary if iCloud backup isn’t activated on the device.

Supported Features:

  • Contacts
  • Call Logs
  • Texts Messages
  • Browser History
  • Events
  • Notes

mSpy Requirements for Android smartphones and tablets

  • The target device must be running Android 4+
  • The target Android device must be connected to the Internet.
  • You need physical access to the device to install mSpy.
  • Instant messengers monitoring work on rooted Androids only.

Determine your Android version

  1. Open your phone's menu. Tap System Settings.
  2. Scroll down towards the bottom.
  3. Select About phone.
  4. The OS version of your device is shown under Android version.

Our computer monitoring software is compatible with Windows XP, Windows 7, Windows 8, Windows 8.1, and Windows Vista.

Determine your Windows version

  1. Go to Start
  2. Type winver into the search field and press Enter.

Our computer monitoring software is Compatible with Mac OS 10.7 Lion, Mac OS X 10.8 Mountain Lion, and OS X 10.10 Mavericks.

Determine your Mac version

  1. Click the Apple menu at the top of your screen.
  2. Select About this Mac.
  3. Your version of OS X will be shown below the letters OS X.
